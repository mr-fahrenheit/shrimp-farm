## Quickstart

```bash
# 1. Toolchain
curl -sSfL https://release.solana.com/stable/install | bash 
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
npm i -g yarn 

# 2. Dependencies
cd shrimp-farm
yarn install 

# 3. Run the tests
anchor test
```
